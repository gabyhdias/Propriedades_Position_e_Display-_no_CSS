Propriedades Position e Display no CSS

Display

Essas palavras-chave especificam o tipo de exibição externa do elemento, que é essencialmente sua função no layout de fluxo.

block

O elemento gera uma caixa de elemento de bloco, gerando quebras de linha antes e depois do elemento quando no fluxo normal.

inline

O elemento gera uma ou mais caixas de elemento em linha que não geram quebras de linha antes ou depois de si mesmas.

inline-block

O elemento gera uma caixa de elemento de bloco que fluirá com o conteúdo ao redor como se fosse uma única caixa em linha.

flex

O elemento se comporta como um elemento de bloco e apresenta seu conteúdo de acordo com o modelo flexbox.

inline-flex

O elemento se comporta como um elemento inline e apresenta seu conteúdo de acordo com o modelo flexbox.

grid

O elemento se comporta como um elemento de bloco e apresenta seu conteúdo de acordo com o modelo de grade.

inline-grid

O elemento se comporta como um elemento inline e apresenta seu conteúdo de acordo com o modelo de grade.

flow-root

O elemento gera uma caixa de elemento de bloco que estabelece um novo contexto de formatação de bloco, definindo onde está a raiz de formatação.

table

Esses elementos se comportam como elementos HTML table. Ele define uma caixa de nível de bloco.

block flow

O elemento ocupa toda a largura disponível, quebra linha antes e depois, e seus filhos se comportam no fluxo normal de documento .

block flex

Define o contêiner como um bloco, mas ativa o Flexbox para os elementos filhos. Os filhos são alinhados em uma linha ou coluna, permitindo fácil alinhamento, distribuição de espaço e responsividade com justify-content e align-items.

block grid

Define o contêiner como um bloco, mas ativa o modelo de layout Grid para os filhos. É bidimensional, permitindo organizar elementos em linhas e colunas simultaneamente, oferecendo controle preciso sobre o posicionamento.

block flow-root

Cria uma caixa de bloco, mas, diferentemente do block comum, ele estabelece um novo contexto de formatação de bloco.

inline flow

O elemento flui como texto, quebra em várias linhas se necessário e ignora width/height.

inline flex

O elemento se comporta como um elemento inline e apresenta seu conteúdo de acordo com o modelo flexbox.

inline grid

O elemento se comporta como um elemento inline e apresenta seu conteúdo de acordo com o modelo de grade.

inline flow-root

Funciona como um bloco na mesma linha, aceitando dimensões e margens normalmente.

Position

A propriedade position do CSS é um recurso que permite organizar os elementos HTML dentro de uma página.

Static

A posição padrão de todos os elementos HTML é a static. Ela é utilizada quando não é necessário alterar o posicionamento do elemento, deixando que ele apareça na mesma ordem em que está inserido no código.

Relative

Ao utilizar o valor relative, é possível reposicionar o componente utilizando os padrões top, button, left e right, sem que esse elemento perca o seu espaço original no layout da página.

Absolute

Um elemento com a configuração position: absolute não faz parte do fluxo normal da página, por isso, ele não tem um espaço próprio no layout e todas as outras estruturas se comportam como se ele não existisse.

Fixed

O valor fixed tem o objetivo de fixar um elemento em um ponto específico da página, permanecendo sempre da mesma forma, não sendo afetado pela barra de rolagem. Além disso, ele também faz com que o item seja completamente removido do fluxo da página.

Sticky

Esse valor combina os comportamentos do relative e do fixed. Isso significa que o elemento com o position: sticky segue o fluxo da página até um determinado ponto, quando passa a ser fixo.
