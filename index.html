Propriedades Position e Display no CSS

POSITION

A propriedade position do CSS é um recurso que permite organizar os elementos HTML dentro de uma página. É esse recurso que possibilita, por exemplo, manter um menu ou um botão sempre visível em uma página, independentemente da posição da barra de rolagem.

Explicação dos valores da propriedade position:

Static: A posição padrão de todos os elementos HTML é a static. Ela é utilizada quando não é necessário alterar o posicionamento do elemento, deixando que ele apareça na mesma ordem em que está inserido no código.

Relative: Ao utilizar o valor relative, é possível reposicionar o componente utilizando os padrões top, button, left e right, sem que esse elemento perca o seu espaço original no layout da página. Essa propriedade é muito usada para organizar movimentos sutis, e que não vão impactar nos demais itens da página.

Absolute: Um elemento com a configuração position: absolute não faz parte do fluxo normal da página, por isso, ele não tem um espaço próprio no layout e todas as outras estruturas se comportam como se ele não existisse.

Fixed: O valor fixed tem o objetivo de fixar um elemento em um ponto específico da página, permanecendo sempre da mesma forma, não sendo afetado pela barra de rolagem. Além disso, ele também faz com que o item seja completamente removido do fluxo da página

Sticky: Esse valor combina os comportamentos do relative e do fixed. Isso significa que o elemento com o position: sticky segue o fluxo da página até um determinado ponto, quando passa a ser fixo. O uso do fixed é muito comum quando é necessário deixar um componente visível por um determinado período, como um menu lateral, por exemplo.



DISPLAY

Block: É uma propriedade CSS que faz com que um elemento HTML ocupe toda a largura disponível (100%), forçando uma quebra de linha antes e depois dele.

Block flow: Define que um elemento ocupe toda a largura disponível (100% do pai) e inicie em uma nova linha, gerando quebras antes e depois dele.

Block flex: Transforma um elemento em um contêiner flexível, organizando seus filhos automaticamente em linha ou coluna.

Block grid: é um sistema de layout bidimensional que organiza elementos em linhas e colunas simultaneamente.

Block flow-root: Cria uma caixa de bloco, mas, diferentemente do block comum, ele estabelece um novo contexto de formatação de bloco.



Inline: Faz com que elementos HTML fiquem um ao lado do outro na mesma linha, ocupando apenas a largura necessária para o seu conteúdo.

Inline-block: O elemento gera uma caixa de elemento de bloco que fluirá com o conteúdo ao redor como se fosse uma única caixa em linha.

Inline-flex: O elemento se comporta como um elemento inline e apresenta seu conteúdo de acordo com o modelo flexbox.

Inline grid: O elemento se comporta como um elemento inline e apresenta seu conteúdo de acordo com o modelo de grade.

Inline flow-root: Funciona como um bloco na mesma linha, aceitando dimensões e margens normalmente.

Grid: Define um elemento como um contêiner de grade, permitindo organizar itens filhos em linhas e colunas estruturadas.

Flex: Transforma um container HTML em um contêiner flexível, alinhando automaticamente seus elementos filhos em uma linha ou coluna.

Flow-root: O elemento gera uma caixa de elemento de bloco que estabelece um novo contexto de formatação de bloco, definindo onde está a raiz de formatação.

Table: Esses elementos se comportam como elementos HTML table. Ele define uma caixa de nível de bloco.
